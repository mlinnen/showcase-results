## Design Decisions
- **ordinal() helper:** Inline function handles edge cases (11th, 12th, 13th)
- **Data structure:** Receives $this->carverData from HtmlView with keys: carver_name, found, results
- **Subtitle logic:**
  - Single-event (year param): "Results for {Event Name} {Year}"
  - Cross-event: "Results across all events"
- **Empty handling:** Sections with no data completely skipped
- **Entry numbers:** 0 or null render as empty `<td></td>`
- **Style values:** N → "Natural", P → "Painted", null → empty cell
- **Semantic HTML:** Uses `<section>`, `<table>`, `<thead>`, `<tbody>`, `<h1>`–`<h4>`
- **CSS classes:** All use cca-* prefix (cca-carver-header, cca-event-section, cca-special-prizes, cca-overall-results, cca-division-results)
